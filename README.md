## 项目介绍
- 项目基于Vue2，自己学习的小例子，往上很多的ego项目没有数据库文件
- **此项目包含数据库文件**
- 数据库文件位置于 server\sql 
- 注意：
- egoVue.sql 是项目所需数据所有的文件   
- project.sql 和userinfo.sql 是胡数据库所需的部分文件，需要写部分逻辑代码的，参考这两个即可
